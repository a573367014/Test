# [稿定编辑器基础库](https://git.gaoding.com/pages/gaoding/editor/docs/gd-eui/)

### install
```
npm i

建议使用 npm i，大仓的部分库如less、less-loader等的版本不符，导致demo没法跑起来，用yarn会导致版本错乱
```

### 调试

```
yarn build
yarn demo
```

### 快速开发指南(*)
```
yarn new
```
**强烈建议使用 yarn new**\
使用yarn new 将会根据开发规范为你自动创建组件、demo、文档。

### 调试文档

```
npm run docs:dev
```
